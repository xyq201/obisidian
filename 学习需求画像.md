# 学习需求画像（用户真实需要什么）

> 本文件驱动「每日 9:00 自动学习」任务。每轮学习后由助手更新，使学习内容越来越贴近用户真实需求：**指导化妆品产品配方设计与开发**。

## 用户身份与目标
- 从事**化妆品 / 护肤产品的研发**，核心是**配方设计与开发**（选料、配比、工艺、稳定性、功效、合规）。
- 关注点不是科普，而是"这个点**怎么指导配方决策**"。

## 已显式关注（来自本对话）
- 原料/活性物：神经酰胺（油溶性、NDS 纳米递送）、屏障修复成分。
- 技术：纳米载体 / 透皮递送、结晶与稳定性难题。
- 落地维度：产品规格（INCI、形态、活性物含量、粒径）、添加量、配方黄金比例（Cer:Chol:FFA = 3:1:1）、配伍/pH/温度、功效验证、供应商（浦瑞 PuriPharm / PuriActives）。

## 当前聚焦方向（初版，待迭代）
1. 皮肤屏障生理学与角质层脂质结构（砖墙模型、渗透路径）。
2. 透皮吸收机理与递送系统（脂质体、纳米乳、包裹）。
3. 活性物规格、配伍、pH、稳定性与工艺温度。
4. 功效评价方法（TEWL、仪器、临床/文献证据）。

## 知识缺口（待深入）
- 主流活性物机制与配方：**烟酰胺（已建基础笔记 2026-07-23，经典文献卷期待补）**；**视黄醇/A醇（已建基础笔记 2026-07-24，SCCS 文号与《规范》限额待核）**；肽类、玻色因、泛醇、维生素 C、水杨酸/果酸等仍待建档案。
  - 烟酰胺可迁移结论：2%–5%、pH 5–7、与原型 VC 错时、内源促神经酰胺合成（与外用神经酰胺协同）。
  - 视黄醇可迁移结论：0.1%–1% 起步低浓度、后段/冷却加入、避光夜间、微囊化降刺激；欧盟 SCCS 限用（身体乳 0.05%RE、其他驻留+淋洗 0.3%RE，2025-11/2027-05 过渡期）。
- 乳化体系与稳定性（O/W、W/O、凝胶、 suspension）、增稠与流变。
- 法规与 INCI、安全用量、宣称合规（各国差异）。
- 消费者趋势与热门宣称的科学支撑。

## 学习优先级原则
1. **经典教材 / 权威综述 / 教科书级来源优先**于碎片文章（公众号热门文章排第二）。
2. 每个知识点必须落到"对配方/开发的指导"：机制 → 痛点/挑战 → 可选原料/方案 → 规格与添加量 → 配伍/pH/稳定性 → 功效评价/法规提示。
3. 少写与配方无关的泛科普；多写可迁移到产品的结论。
