# 每日学习摘要 · 2026-07-23

## 今日主题
**烟酰胺（Niacinamide / 维生素 B3）** —— 主流"百搭"活性物的配方档案（机制 → 痛点 → 规格/添加量 → 配伍/pH → 功效/法规），并与既有神经酰胺笔记做"内源促生 + 外源补充"桥接。

## 关键知识点（结论 + 配方启示）
1. **烟酰胺是水溶性维生素 B3 酰胺，温和无光敏**；配方启示：可加在水相、耐热、非热敏，工艺宽容。
2. **核心机制之一是上调角质层神经酰胺合成** → 降低 TEWL、增厚屏障；配方启示：与外用神经酰胺（3:1:1 黄金比例）**双路径协同修护屏障**。
3. **淡斑走"抑制黑素小体转移"而非抑制酪氨酸酶**；配方启示：与 VC/377/曲酸等酪氨酸酶路径成分可错峰叠加，覆盖不同靶点。
4. **常用 2%–5%，屏障/色沉临床多取 4%–5%**；配方启示：普通保湿 2% 起，功效精华 4%–5%，敏感肌低浓度起步防刺激。
5. **推荐 pH ~5.0–7.0，避免强碱长期以防水解生成烟酸（致潮红/刺痛）**；配方启示：控 pH + 选高纯低烟酸杂质原料。
6. **与原型 VC（L-ascorbic acid）宜错时**：VC 需 ~pH3，二者高温低 pH 可能相互转化；配方启示：早晚分开，或改用 VC 衍生物（SAP/MAP）。
7. **控油缩毛孔、抗炎舒缓、抗黄抗皱**多效一体，且兼容视黄醇/肽/HA/泛醇；配方启示：适合做"全能基底活性"复配。
