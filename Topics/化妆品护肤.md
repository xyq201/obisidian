# 化妆品护肤（MOC）

> 原料、配方、递送、稳定性、功效与合规。聚焦"可指导产品配方设计与开发"的知识点。

## 笔记清单
- [神经酰胺（油溶性 · PuriActives® / 浦瑞医药）](原料学习知识/神经酰胺-油溶性-PuriActives.md) — 屏障成分；NDS 纳米递送；规格与配方黄金比例 3:1:1
- [烟酰胺（Niacinamide / 维生素 B3）](原料学习知识/烟酰胺-Niacinamide.md) — 百搭活性；屏障(促内源神经酰胺)/控油/淡斑；2%–5%，pH 5–7，与原型 VC 错时
- [视黄醇（Retinol / A醇 / 维生素A）](原料学习知识/视黄醇-Retinol.md) — 抗衰黄金标准；类视黄酸促胶原；不稳定+刺激为配方核心难点；欧盟 SCCS 限用(待核文号)

- 配伍 / pH / 工艺温度
