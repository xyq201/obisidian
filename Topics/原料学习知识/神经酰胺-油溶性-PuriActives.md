# 神经酰胺（油溶性 · PuriActives® / 浦瑞医药）

> 学习笔记：从一篇品牌技术推文提炼的成分档案。侧重「是什么 / 为什么难用 / 怎么解决 / 怎么用」。

## 1. 基础认知
- **是什么**：神经酰胺（Ceramide）是鞘脂，由神经鞘氨醇长链碱基 + 脂肪酸组成；为角质层脂质主要成分，约占 40%–50%。
- **砖墙结构**：与胆固醇（25%）、游离脂肪酸（15%）共同构成"砖墙结构"中的"灰浆"，形成多层脂质屏障。
- **黄金比例**：Cer : 胆固醇 : 游离脂肪酸 = **3 : 1 : 1**。
- **其他事实**：人体皮肤中神经酰胺种类 12+；角质层脂质中神经酰胺占比约 50%。

## 2. 应用三大难题（传统配方痛点）
- **高结晶性**：熔点高、溶解度低，配方中易结晶析出形成可见颗粒，高浓度时更突出。
- **低生物利用度**：以悬浮态存在，难穿透角质层；即便有油酸辅助也无法有效渗透。
- **配伍性差**：水/油溶性都有限，与多种体系兼容性不佳，工艺复杂。

## 3. NDS 纳米传输系统（浦瑞专有）
- 技术：纳米脂质体包裹，将神经酰胺稳定包裹于纳米级磷脂双分子层载体，**粒径 <30nm**。
- 三大优势：高效透皮（提高皮肤滞留量与生物利用度）、靶向递送（毛囊靶向 + 储库作用，持续释放）、稳定保护（隔绝氧化与结晶析出）。

### 纳米 vs 传统对比
| 维度 | 传统配方 | NDS 纳米传输 |
|---|---|---|
| 粒径 | >1000nm（悬浮/微乳） | <30nm（透明纳米浓缩液） |
| 透皮效率 | 低，难穿角质层 | 高，角质层定位提升 **9 倍** |
| 稳定性 | 易结晶、温度敏感 | 热/机械/pH 稳定俱佳 |
| 溶解性 | 水油均差 | 既可水溶也可油溶 |
| 配伍性 | 有限 | 适合各种体系 |

## 5. 核心功效
- **屏障修复**：补充角质层脂质、修复砖墙结构、降低 TEWL。
- **深层保湿**：增强角化细胞间粘着力、长效锁水、改善干燥脱屑。
- **抗衰老**：增厚角质层、增强弹性、减少细纹。
- **舒缓防护**：抑制有害物侵入、降低过敏，适合敏感/受损肌及医美术后。

## 6. 配方指南（速查卡片）
- 黄金比例 **3 : 1 : 1**（神经酰胺 : 胆固醇 : 游离脂肪酸）。
- 建议添加量 **0.1%–3.0%**；可在**水相或冷却阶段**加入。
- 兼容大多数化妆品原料（含高浓度乙醇、电解质）。
- 加工温度 **≤80°C**；终产品 pH **3.0–7.0**；稳定温度 **4–25°C**；避免阳光直射。
- 适用品类：面部护理（屏障修复精华/抗衰面霜/敏感肌乳液）、身体护理、头皮护理（去屑止痒洗发水等）、婴童护理。
