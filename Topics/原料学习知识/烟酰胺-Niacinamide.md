# 烟酰胺（Niacinamide / 维生素 B3 / Nicotinamide）

> 学习笔记：主流"百搭"活性物。侧重「机制 → 配方痛点 → 规格/添加量 → 配伍/pH/稳定性 → 功效/法规」，并与已学的神经酰胺笔记做桥接。

## 1. 基础认知
- **是什么**：维生素 B3 的酰胺形式（烟酸的酰胺化合物），白色结晶性粉末；**水溶性**、无光敏性。INCI 名称：**Niacinamide**。
- **与烟酸区别**：烟酸（nicotinic acid）是酸形式，高浓度易致血管扩张/潮红；烟酰胺为酰胺，温和不潮红——但**原料中烟酸杂质需严控**。
- 人体不能足量自产，靠膳食与护肤补充。

## 2. 作用机制（落到配方启示）
- **辅酶前体**：转化为 NAD⁺/NADP⁺，参与数百项氧化还原反应，支撑细胞能量与生物合成。
- **屏障（关键桥接）**：上调角质层**神经酰胺（鞘脂）、游离脂肪酸、胆固醇**合成 → 增厚屏障、降低 TEWL。→ 与外用神经酰胺（见 [[神经酰胺-油溶性-PuriActives]]）形成"**内源促生 + 外源补充**"双路径协同。
- **色素**：抑制黑素小体从黑素细胞向角质形成细胞**转移**（非抑制酪氨酸酶）→ 均匀肤色、改善色沉。
- **皮脂**：下调皮脂分泌速率 → 改善油光与毛孔外观。
- **抗炎/抗氧化**：抑制 ROS、降低 IL-1 等炎症因子；中和自由基。
- **其他**：改善红斑、皮肤黄化（sallowness）、细纹。

## 3. 配方应用痛点
- **纯度/杂质**：含烟酸杂质 → 刺痛、潮红、灼热（高浓度/敏感肌更明显）。→ 选高纯原料 + 控 pH。
- **pH 水解**：极端 pH（尤其强碱，或强酸+高温长时间）下可水解生成烟酸。与 **L-抗坏血酸（原型 VC）在 low pH + 高温**共存时可能相互转化、降低效用——常错时/分区使用。
- **高浓度刺激**：>5% 个别出现刺激；敏感肌宜从低浓度起步。
- **配伍整体宽容**：与多数体系兼容；强酸体系（pH<3.5 的 VC 精华）配方上宜分离。

## 4. 产品规格与添加量（配方速查）
| 项目 | 值/说明 |
|---|---|
| INCI | Niacinamide |
| 形态 | 白色粉末（亦有预溶溶液/复方） |
| 溶解性 | 水溶性（常配成水溶液或甘油分散） |
| 建议添加量 | **2%–5%** 常用；屏障/色沉临床常 **4%–5%**；控油 2%–4% |
| 工艺 | 水相加入；**耐热**，可高温阶段（非热敏） |
| 推荐 pH | **~5.0–7.0**（皮肤生理 pH 附近）；避免强碱（>~8）长期以防水解 |
| 上限 | 依配方与地区法规，一般 ≤5% 安全；更高需论证 |

## 5. 配伍矩阵
- ✅ **协同/兼容**：视黄醇/视黄醛、肽类、肌肽、透明质酸、泛醇、温和酸类（乳糖酸/PHA）、**神经酰胺（屏障协同）**、甘油等保湿剂。
- ⚠️ **注意**：原型 VC（L-ascorbic acid）——pH 需求冲突（VC 需 ~pH3），且与烟酰胺高温低 pH 可能相互转化；建议**早晚错时**或改用 VC 衍生物。属配方 pH/稳定策略问题，非绝对禁忌。
- 总体：宽容度高，是配方中的"百搭"活性。

## 6. 功效与证据（待按需深挖）
- 屏障/降 TEWL、控油缩毛孔、均匀肤色/淡斑、抗皱抗黄、舒缓——多项人体临床支持。代表：**Bissett et al. 2006, Int J Cosmet Sci, PMID 17147561**（4%–5% 使用 12 周改善皮肤黄化、皱纹、红斑、色沉）。
- 机制层面（上调神经酰胺、抑制黑素小体转移）为学界共识；经典原始文献（如 Tanno 2000 屏障、Hakozaki 2002 色素转移）**具体卷期待核实**，建议用户补 PubMed 原文后再写入精确引用。

## 7. 功效评价（研发可用）
- 屏障：TEWL；控油：皮脂分泌率/Sebumeter；色素：色度、Mexameter、数码影像；皱纹/红斑：影像评级 + 临床评分。

## 8. 法规提示
- 多地区允许，作为安全活性广泛用于护肤。最高用量遵循当地规定（中国《已使用化妆品原料目录》、EU SCCS、FDA），**2%–5% 属常见合规区间**；宣称需有功效评价依据。
